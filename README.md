# Target Citizens Dashboard

Тестовый frontend-проект на `React + TypeScript + Vite`.

## Текущий стек

- `React`
- `TypeScript`
- `Vite`
- `React Router DOM`
- `ESLint`
- `Prettier`

## Команды

```bash
npm install
npm run dev
```

Сборка клиента:

```bash
npm run build
```

Проверка линтера:

```bash
npm run lint
```

Форматирование:

```bash
npm run format
```
